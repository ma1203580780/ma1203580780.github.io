nginx配置跳转的一个实例

：-}

需求

目前移动端的域名为 https://m.yourdomain.com ,为了防止某个活动页面在微信内被封导致整个移动端都不能访问，所以新配置了一个域名，叫 https://h5.yourdomain.com 。

配置规则

https://m.yourdomain.com/expand/index 跳转  https://h5.yourdomain.com/expand/index 
https://h5.yourdomain.com/*（除了/expand/index）跳转  https://m.yourdomain.com/*

配置文件

m.conf部分配置

location / {
root /var/www/m-julyedu/dist;
index index.html;
location / {
try_files $uri $uri/ /index.html;
}
}
location ^~ /expand/index {
rewrite ^/(.*)$ https://active.julyedu.com/$1 break;
}
