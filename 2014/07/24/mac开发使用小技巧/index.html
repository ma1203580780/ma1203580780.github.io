mac系统快捷操作笔记（二）

把文件创建快捷方式到桌面，用软链接

ln -s

apache虚拟主机配置在/etc/apache2/extra/httpd-vhosts.conf

然后配置本身的/etc/hosts

127.0.0.1       chaoniu.com
127.0.0.1       www.super.com

apache开启重写 ，在/etc/apache/httpd.conf

打开注释：

LoadModule rewrite_module libexec/apache2/mod_rewrite.so

配置虚拟主机

<VirtualHost *:80>
    ServerAdmin webmaster@dummy-host.example.com
    DocumentRoot "/www/chaoniu/public/"
    ServerName www.super.com
    ServerAlias www.dummy-host.example.com
    ErrorLog "/private/var/log/apache2/www.1.com-error_log"
    CustomLog "/private/var/log/apache2/www.1.com-access_log" common
</VirtualHost>


重启apache：sudo /usr/sbin/apachectl restart

关闭apache：sudo /usr/sbin/apachectl stop

开启apache：sudo /usr/sbin/apachectl start

安装Vue：

brew install node
npm -v
npm install -g vue-cli
