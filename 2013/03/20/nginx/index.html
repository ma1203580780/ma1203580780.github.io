nginx

nginx入门

：-}

安装nginx

brew search nginx
brew install nginx

启动nginx

sudo nginx

访问localhost:8080 发现已出现nginx的欢迎页面了。

做个软连接

ln -s  /usr/local/sbin/nginx /usr/bin/nginx

常用的指令有：

nginx -V 查看版本，以及配置文件地址
nginx -v 查看版本
nginx -c filename 指定配置文件
nginx -h 帮助

#重新加载配置|重启|停止|退出 nginx

nginx -s reload|reopen|stop|quit

#测试配置是否有语法错误

nginx -t

查看nginx的进程pid，并杀死进程，但不要杀死父进程。

ps -ef | grep nginx

如果杀死父进程。要执行这条命令，加载nginx.conf文件

nginx -c /usr/local/etc/nginx/nginx.conf
